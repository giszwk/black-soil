# 大数据驱动的耕地质量演变预测模型和产量估测模型 — 项目总结

---

## 一、一句话概括

> 本项目构建了两套 AI 模型——用 **LSTM 深度学习**预测耕地质量变化，用 **GWR 地理加权回归**估测粮食产量，并将两者耦合，揭示"地好才能产高"的定量关系，服务于东北黑土地保护和国家粮食安全。

---

## 二、项目背景与意义

### 2.1 为什么重要

东北黑土地是世界四大黑土区之一，贡献了全国 **1/4 的粮食产量**和 **1/3 的商品粮**，是国家粮食安全的"压舱石"。然而黑土地正面临严重威胁：

- **北部**（松嫩平原北部）：气候冷凉、土壤酸化、春旱频发
- **中部**（旱田坡耕地）：水蚀严重，水田土壤粘重、养分有效性差
- **南部**（瘠薄黑土区）：土壤有机质低、耕层浅薄

### 2.2 现有方法的问题

| 问题 | 说明 |
|------|------|
| 影响因子单一 | 主要依赖气象和遥感数据，缺乏田间实测数据 |
| 方法传统 | 深度学习等新方法应用不足 |
| 缺少动态预测 | 多为静态评估，无法近实时监测变化 |
| 质量-产量脱钩 | 耕地质量评价与产量估测独立进行，关系不清晰 |

### 2.3 本项目目标

依托**多要素联网观测大数据**，构建耕地质量演变预测与粮食产量估测的耦合模型，明确质量与产能的响应机制，为松嫩平原黑土区耕地保护与可持续利用提供科学工具。

## 三、技术方法

### 3.1 整体框架

```
多源大数据（气象、土壤、遥感、人文）
        │
        ├──→ LSTM 耕地质量演变预测模型 ──→ 耕地质量评价结果
        │                                          │
        │                                          ▼
        └──→ GWR 粮食产量估测模型 ←────── 耦合耕地质量结果
                        │
                        ▼
                产量估测 + 显著性检验
```

### 3.2 模型一：LSTM 耕地质量演变预测模型

**核心思路**：利用长短期记忆神经网络（LSTM）处理时序数据的能力，对耕地质量进行近实时动态预测。

**输入数据（四大类关键限制因子）**：

| 类别 | 具体指标 |
|------|----------|
| 气象数据 | 降水量、温度 |
| 地形数据 | 数字高程（DEM） |
| 土壤数据（实地采样） | 有效土层厚度、有机质含量、养分状况、土壤质地、障碍因素、容重、耕层厚度、黑土层厚度、pH值、电导率、土体构型、含水量、全氮、有效磷、速效钾、水土保持系数 |
| 人文指标 | 耕地投入、灌溉条件、连片度 |

**模型结构**：输入层 → LSTM层（遗忘门、输入门、输出门） → 全连接层 → 质量评估输出

**训练策略**：60% 训练集、20% 验证集、20% 测试集，采用 MSE/RMSE 评估，交叉验证优化。

### 3.3 模型二：GWR 粮食产量估测模型

**核心思路**：利用地理加权回归（Geographically Weighted Regression）捕捉产量在空间上的异质性，并**耦合耕地质量演变预测结果**。

**输入数据**：

- 气象数据：降水量、温度、风速、蒸散、全年日照时数
- 土壤数据：有机质、养分、质地、容重、耕层厚度、黑土层厚度、pH值
- 人文数据：播种面积、有效灌溉率、农业劳动力、农机投入、农药和薄膜使用量
- **耕地质量演变预测结果**（来自模型一）

**关键技术环节**：

1. 构建空间权重矩阵（欧几里得距离 + 最近邻法）
2. GWR 回归分析
3. 参数估计与显著性检验（最小二乘法 + t检验）

### 3.4 技术路线示意

```
多要素联网观测大数据
    ┌────┴────┐
    │         │
特征提取   空间权重矩阵
    │         │
LSTM层      GWR回归
    │         │
全连接层   参数检验
    │         │
    └────┬────┘
         │
   质量与产量耦合结果
         │
   可视化与决策支持
```

## 四、当前进展（截至 2026 年 3 月）

### 4.1 模型构建

- **试验区域**：黑龙江省北安市、方正县 + 吉林省农安县（松嫩平原黑土示范区）
- **数据来源**：实地采样土壤数据 + 遥感影像（Landsat 8、NASA DEM）+ 气象数据 + 统计数据
- **已完成**：耕地质量演变预测模型和粮食产量估测模型各 **1 套**，初步验证有效

### 4.2 初步成果（以方正县为例）

**耕地质量评估**：模型能够有效整合多源数据，反映耕地质量关键演变特征，实现动态监测。

**产量估算**：模型捕捉到粮食产量在空间分布上的异质性，气象要素与遥感特征对作物生长的综合影响得到量化。

## 五、关键技术优势

1. **多因子融合**：区别于传统仅用气象和遥感数据的方法，本项目融合气象、土壤、地形、人文四大类指标，模型更可靠
2. **动态预测**：LSTM 模型具备时序记忆能力，可实现近实时动态监测而非单次静态评估
3. **质量-产量耦合**：首次将耕地质量演变预测结果显式接入产量估测模型，明确两者的定量关系
4. **空间异质性**：GWR 模型通过空间权重矩阵捕捉不同区域的差异性，结果更符合真实农业生产格局

## 六、当前局限与下一步计划

### 6.1 当前局限

| 局限 | 说明 |
|------|------|
| 数据量稀缺 | 采样点主要集中在 3 个县市，样本点过少 |
| 关键指标不明确 | 哪些因子对质量/产量贡献最大，仍需进一步分析 |
| 缺少未来情景预测 | 当前模型主要基于历史/现状数据，尚未接入未来气候情景 |

### 6.2 下一步计划

1. **引入未来气候情景数据**（IPCC 共享社会经济路径下的温度、降水预测）
2. **结合不同管理策略**（保护性耕作、有机肥替代等）开展情景模拟
3. **短期滚动预报**：耦合气象预报与遥感监测数据，实现高分辨率产量分布预测
4. **模型参数优化**：补充更多实测数据，提升泛化能力
5. **提供决策支持**：产能预警、分区调控、精准施肥灌溉指导

## 七、预期效益

- **学科效益**：推动精准农业与智慧农业的学科交叉，实现耕地质量从"人工抽样调查"向"空天地一体化智能感知"的范式转变
- **社会效益**：提升黑土地监测保护水平，服务国家粮食安全战略
- **经济效益**：通过精准预测引导肥料和灌溉投入，减少资源浪费，优化种植规划
