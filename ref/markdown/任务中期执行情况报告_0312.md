# 二、取得的重要进展及成果

## 1、任务中期重要进展及成果

耕地质量/粮食产能双提升与国家粮食安全息息相关，而耕地质量/粮食产能受到土壤肥力、水土保持性状、农田作物类型及利用方式、气象条件等多种影响要素的影响，因此急需结合多影响因子的耕地质量/粮食产量评估预测方法。本子课题利用多要素联网观测大数据，结合作物生长条件和环境的空间异质性，构建耕地质量演变预测和粮食产量估测模型，以量化环境特征，明确耕地质量与粮食产量的内在联系，为农田粮食产能的精准评估提供支持。本子课题已初步建立松嫩平原黑土区耕地质量预测模型、粮食产量估测模型各1套，并基于上述模型申请专利1项，发表论文1篇，正在送审阶段论文1篇，具体如下：

（1）新方法：大数据驱动的耕地质量演变预测模型和产量估测模型，该方法明确耕地质量与粮食产量的耦合关系，助力于黑土地保护与农业数字化；

（2）发明专利：耕地产量预测方法、装置、电子设备及存储介质，已进入实质审查阶段；

（3）高水平论文：发表1篇（Improving land surface phenology extraction through space-aware neural networks），该成果辅助更精确地模拟和预测耕地质量关键指标（如有机质、含水量）的动态演变过程；送审1篇（Mapping 20-years winter wheat dynamics in global primary planting areas using Gaussian mixture models in a flexible way），该成果为本项目分析耕地质量演变及其对粮食产量的影响提供长期、可靠的时空本底数据。

随着农业数据化不断推进，基于大数据和机器学习的管理模式愈发普及，结合深度学习与地理加权回归的耕地质量预测和粮食产量估测模型充分利用多源数据，为农业生产提供数据支持与政策指导。该专利能够优化耕地种植机构，提升土壤管理措施，为耕地施肥与灌溉提供辅助建议，从而促进农田长期稳定生产与可持续发展，提高农业现代化水平，服务于国家粮食安全的大战略。

## 2、预期经济社会效益

**学科/行业效益**：本课题深度融合农业科学、遥感技术与人工智能，构建基于深度学习的耕地质量与作物产量预测模型，将有力推动精准农业与智慧农业的学科交叉发展。一方面，将遥感大数据与深度学习算法结合，可实现耕地质量从“人工抽样调查”向“空天地一体化智能感知”的范式转变；另一方面，作物产量估测模型的应用将推动农业气象、土壤学和作物生理生态模型的数字化升级，为农业遥感领域提供新的理论与方法支撑。此外，该成果可带动农业信息化服务和智能装备制造的发展，助力我国农业现代化水平的整体提升。

**社会效益**：本子课题申请专利克服了大尺度的耕地质量评价和粮食产能预测主要依赖于气象和遥感数据，在考虑影响因子方面相对单一的缺陷，填补了缺乏多因子多模型的耕地质量和粮食产能耦合评估模型的空白；通过相关技术的推广，能够提升黑土地的监测与保护水平，带动与引领农业遥感相关产业的升级与调整，提高农业现代化水平，服务与国家粮食安全的大战略；通过相关技术的示范应用，指导相关农技精准实施，提高黑土地的利用与保护水平，降低黑土地因侵蚀而造成的环境风险。

**经济效益**：通过耕地质量的动态预测能够更精确地引导肥料与灌溉等的资源投入，减少不必要的资源浪费，优化种植结构与土壤管理措施，延长耕地的使用寿命，保障长期的农业收益和土地资源的可持续利用。通过粮食产量估测模型，政府与农业企业能够根据产量估测结果合理优化种植规划，提升资源配置效率，科学安排生产、储存和流通，降低经济损失。

**人才、专利与技术**：项目期内已培养掌握跨学科知识的复合型人才3名，其中博士生1名；围绕模型算法、数据处理等核心环节申请发明专利1项；开发大数据驱动的耕地质量演变预测模型和产量估测模型新方法，为数字农业的高质量发展奠定基础。
