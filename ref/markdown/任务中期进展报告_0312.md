# 3.任务取得的重要进展及成果

## 3.1任务中期重要进展

### 3.1.1 总体进展

耕地质量/粮食产能双提升与国家粮食安全息息相关，而耕地质量/粮食产能受到土壤肥力、水土保持性状、农田作物类型及利用方式、气象条件等多种影响要素的影响，因此急需结合多影响因子的耕地质量/粮食产量评估预测方法。本子课题组基于大数据、机器学习与地理加权回归等方法构建了耕地质量预测和粮食产量估测模型，充分利用多源数据，为农业生产提供数据支持与政策指导，服务于国家粮食安全的大战略。于2023年至今已初步建立松嫩平原黑土区耕地质量预测模型、粮食产量估测模型各1套，并基于上述模型申请专利1项，发表论文1篇，正在送审阶段论文1篇。

### 3.1.2 具体研究进展

由于采样点数据主要集中在北安市、方正县和农安县，模型初步构建试验地区选择为包括黑龙江省北安市、方正县以及吉林省的农安县的松嫩平原黑土示范区。数据选取上，除示范区实地采样获取的土壤性质数据和产量数据外，本子课题将从遥感与统计手段出发，获取相关数据集，具体包括遥感影像数据集和遥感反演数据。其中遥感影像数据集主要包括降水、温度、风速、高程，遥感反演数据集含土壤水分数据集等。

依托上述多要素联网观测获取的大数据，本模型针对不同耕地自然及人为作用条件下的质量评价关键及限制因子，利用机器学习重构多个影响耕地质量（土壤、气象、水文等）关键变量的近实时预测模型，实现基于大数据驱动、联网监测的耕地质量演变预测模型的构建。接着，通过地理加权回归等数据驱动模型，明确耕地质量与产能提升的耦合关系，构建不同耕地质量条件下的粮食产量估测模型（见图1）。

图6. 专利申请进入实质审查阶段通知书

东北黑土地是全球四大黑土区之一，性状好、肥力高、适宜作物生长的优质土地。粮产量是我国粮食总量的1/4、全国商品粮的1/3，是国家粮食安全的压舱石。但如今黑土地面临土壤酸化、盐渍化、水蚀严重、有机质降低等严重的威胁，因而迫切需要精确监测耕地质量/粮食产量的方法，助力黑土地保护以及国家粮食安全战略。

现有大尺度耕地质量评价和粮食产量预测主要依赖于气象和遥感数据，较少结合田间数据，缺乏多因子耦合评估模型。而实际上，耕地质量/粮食产能受到土壤肥力、水土保持性状、农田作物类型等多种影响要素的影响。因此，本专利引入多要素联网观测获取大数据开展多因子建模。为了解决当前产量预测模型单一且效果不佳，深度学习等新方法较少应用、缺少动态预测的问题，本专利采用多模型综合建模预测未来结果。同时，过往研究多未明确耕地质量与产量估计的提升关系。为此，本专利耦合了耕地质量与粮食产量，明确两者内在联系，推动耕地质量和粮食产量的双提升。

本成果通过专利技术的推广，填补了缺乏多因子多模型耕地质量与产能耦合评估的空白，推动黑土地监测与保护，为农业生产和政策制定提供数据支持，提升农业现代化水平，助力国家粮食安全战略；通过动态预测耕地质量，可精准引导肥料和灌溉投入，减少资源浪费，保障农业长期收益和土地资源可持续利用；通过粮食产量估测模型可帮助政府和农业企业优化种植规划，提升资源配置效率，降低经济损失。
