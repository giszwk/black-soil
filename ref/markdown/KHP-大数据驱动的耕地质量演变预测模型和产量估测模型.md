# 大数据驱动的耕地质量演变预测模型和产量估测方法

技术领域

本发明涉及数据科学和农业科学的交叉领域，尤其涉及农业生产与耕地保护，是一种依托大数据驱动的耕地质量演变预测与产量估测方法。

背景技术

东北黑土地是全球四大黑土区之一，是国家粮食安全的压舱石。但目前松嫩平原北部厚层黑土玉豆轮作区气候冷凉、土壤酸化、春旱频发；中部旱田坡耕地水蚀严重，水田土壤粘重，养分有效性差；南部瘠薄黑土区，土壤有机质低、耕层浅薄。针对上述问题，急需耕地质量/粮食产能的评估预测方法，为松嫩平原耕地质量与粮食产能双提升技术模式构建提供支，助力国家粮食安全。

但目前大尺度的耕地质量评价和粮食产能预测主要依赖于气象和遥感数据，在考虑影响因子方面相对单一，深度学习等新方法运用较少，缺少动态预测，并且未将耕地质量与产量估计建模结合，耕地质量和粮食产量提升关系不清晰。

发明内容

本发明提供了一种基于大数据的耕地质量演变预测与粮食产能估测方法，用以明确耕地质量与产能提升的耦合关系，构建松嫩平原不同区域耕地质量和粮食产能双提升技术模式，开展科技支撑的耕地质量与粮食产能双提升技术模式效果评价，服务于黑土地保护工程，助力国家粮食安全。

根据本发明提供的一种基于大数据的耕地质量演变预测方法，将所述带有长期依赖关系的特征信息输入全连接层，输出对耕地质量的评估结果，包括：将所述带有长期依赖关系的特征信息输入，对输入进行线性变化，即权重矩阵相乘，并添加上偏置向量；将所述线性变化结果通过激活函数进行处理，输出耕地质量的评估结果。
